🚗 Hướng Dẫn Cực Chi Tiết: Setup Project Car Rental

Dành cho thành viên chưa cài đặt bất cứ phần mềm nào trên máy.

☕ 1. Cài Đặt Java (JDK 21) & Cấu Hình Biến Môi Trường

Project này bắt buộc phải dùng Java 21.

  1. Truy cập Adoptium JDK 21. Tải file cài đặt đuôi .msi hoặc .exe dành cho Windows x64.
  2. Chạy file vừa tải, cứ nhấn Next liên tục cho đến khi Finish. Cứ để nó cài vào đường dẫn mặc định (thường là C:\Program Files\Eclipse Adoptium\jdk-21... hoặc C:\Program Files\Java\jdk-21...).
  3. Bước Quan Trọng Nhất: Cài biến môi trường (PATH) để máy tính hiểu lệnh Java:
    • Nhấn phím Windows (biểu tượng cửa sổ trên bàn phím), gõ tìm kiếm chữ: Environment Variables. Chọn kết quả "Edit the system environment variables".
    • Một cửa sổ nhỏ hiện ra, bạn nhấn nút "Environment Variables..." ở góc dưới bên phải.
    • Ở bảng phía dưới cùng (System variables), cuộn tìm dòng chữ Path. Click chuột trái vào chữ Path đó rồi nhấn nút Edit...
    • Nhấn nút New. Dán đường dẫn thư mục bin của Java vào. (Ví dụ: C:\Program Files\Eclipse Adoptium\jdk-21.0.2.13-hotspot\bin).
    • Tiếp tục ở bảng System variables, nhấn New... để tạo biến mới:
      Variable name: JAVA_HOME
      Variable value: Điền đường dẫn thư mục Java nhưng BỎ thư mục bin đi. (Ví dụ: C:\Program Files\Eclipse Adoptium\jdk-21.0.2.13-hotspot).
    • Nhấn OK ở tất cả các bảng để lưu lại.
  4. Kiểm tra xem đã thành công chưa: Nhấn phím Windows + R, gõ cmd rồi Enter. Gõ lệnh:
    java -version
    Nếu nó hiện ra chữ "openjdk version 21..." thì xin chúc mừng, bạn đã làm đúng!

🐙 2. Cài Đặt Git

  1. Truy cập Git for Windows và tải bản 64-bit Git for Windows Setup.
  2. Chạy file vừa tải. Lúc cài đặt màn hình sẽ hỏi rất nhiều thứ lằng nhằng, bạn KHÔNG CẦN CHỈNH GÌ CẢ, cứ nhắm mắt nhấn Next liên tục đến khi Install và Finish.

🐱 3. Cài Đặt Apache Tomcat 10.1

Chú ý: Bắt buộc phải là Tomcat bản 10.x trở lên. Không dùng bản 9 vì dự án này dùng công nghệ Jakarta EE.
  1. Truy cập Trang tải Tomcat 10.
  2. Trong phần Core, nhấp vào link 64-bit Windows zip để tải về.
  3. Sau khi tải xong, click chuột phải vào file zip, chọn Extract All... (Giải nén).
  4. NÊN giải nén ra thẳng ổ C để đường dẫn ngắn gọn, dễ tìm. Ví dụ: C:\apache-tomcat-10.1.xx. Tuyệt đối không để trong thư mục có dấu tiếng Việt (ví dụ `C:\Thư mục của tôi`).

🗄️ 4. Cài Đặt & Cấu Hình SQL Server (Quan Trọng 🌟)

  1. Cài đặt SQL Server Developer: Tải bộ cài SQL Server trên trang chủ Microsoft, chọn kiểu cài đặt Basic hoặc Custom. Tại bước xác thực (Authentication Mode), PHẢI CHỌN "Mixed Mode" và nhập mật khẩu cho tài khoản sa (ví dụ nhập mật khẩu là 123 cho dễ nhớ).
  2. Cài đặt SSMS: Tải SQL Server Management Studio (SSMS) và cứ Next để cài đặt.
  3. BẬT KẾT NỐI MẠNG CHO SQL SERVER (Bước hay bị quên nhất):
    • Nhấn phím Windows, tìm và mở app SQL Server Configuration Manager.
    • Ở cột bên trái, bấm nút mũi tên sổ xuống chỗ SQL Server Network Configuration -> Chọn Protocols for MSSQLSERVER.
    • Nhìn sang cột bên phải, thấy dòng TCP/IP đang bị tắt (Disabled). Click chuột phải vào nó chọn Enable.
    • Tiếp tục click chuột phải vào TCP/IP chọn Properties.
    • Chuyển sang tab IP Addresses. Cuộn thanh trượt xuống tuốt dưới cùng, tìm phần có tên là IPAll.
    • Xóa trống ô TCP Dynamic Ports (nếu có số 0 thì xóa đi). Ở ô TCP Port, nhập số 1433. Nhấn OK.
  4. Khởi động lại SQL Server: Nhấn phím Windows, gõ Services. Tìm dòng SQL Server (MSSQLSERVER), chuột phải chọn Restart.

📥 5. Kéo Source Code (Clone) và Khởi Tạo Database

A. Tải Source Code

Mở thư mục bạn muốn lưu dự án (ví dụ `Documents`). Click chuột phải vào khoảng trống chọn Open in Terminal (hoặc Git Bash Here).

Gõ lệnh sau và Enter:

git clone [LINK_GITHUB_CỦA_NHÓM_BẠN]

Sau khi chạy xong sẽ có thư mục car-rental-management.

B. Khởi tạo Database

  1. Mở phần mềm SQL Server Management Studio (SSMS). Đăng nhập bằng `Authentication: SQL Server Authentication`, Login: sa, Password: 123 (hoặc pass bạn đã đặt).
  2. Vào thư mục car-rental-management\database bạn vừa clone về. Bạn sẽ thấy 2 file: schema.sql và sample-data.sql.
  3. Kéo thả file schema.sql vào cửa sổ SSMS. Bấm nút Execute (F5). Chờ nó báo Success.
  4. Tiếp tục kéo thả file sample-data.sql vào SSMS và bấm Execute (F5).

👉 Vậy là DB của bạn đã sẵn sàng với 5 tài khoản và 10 chiếc ô tô xịn xò.

IDE 6. Cấu Hình Project Trong Apache NetBeans

  1. Mở NetBeans. Trên thanh menu, chọn File -> Open Project... và chọn thư mục car-rental-management. Đợi một xíu dưới góc phải báo "Background scanning" chạy xong.
  2. Thêm Tomcat vào NetBeans:
    • Nhìn sang panel bên trái, bấm vào tab Services (cùng hàng với chữ Projects).
    • Chuột phải vào hình bánh răng Servers -> Add Server...
    • Chọn Apache Tomcat or TomEE -> Next.
    • Bấm nút Browse... và chỉ đường dẫn tới thư mục C:\apache-tomcat-10.1.xx mà bạn đã giải nén ở Phần 3.
    • Nếu nó bắt nhập user/pass thì cứ nhập bừa admin/admin. Nhấn Finish.
  3. Đổi Mật Khẩu Database:
    Quay lại tab Projects. Mở thư mục Source Packages -> (default package). Click đúp mở file db.properties.
    Nếu mật khẩu SQL của bạn là 123, hãy đảm bảo nội dung file là:
    db.user=sa
    db.password=123
    Đừng quên bấm Ctrl + S để lưu lại.

🚀 7. Chạy Ứng Dụng Nào!

  1. Click chuột phải vào tên Project (có hình ly cà phê và chữ m màu vàng) -> Chọn Clean and Build (Shift + F11).
    Lưu ý: Lần chạy đầu tiên sẽ tốn khoảng 1-2 phút vì Maven phải tải các thư viện xuống máy. Vui lòng kiên nhẫn đợi đến khi thấy chữ BUILD SUCCESS màu xanh lá cây.
  2. Sau khi Build Success, nhấn chuột phải vào tên Project -> Chọn Run (F6).
  3. Trình duyệt sẽ tự bật lên hoặc bạn tự vào link: http://localhost:8080/car-rental-management/home
